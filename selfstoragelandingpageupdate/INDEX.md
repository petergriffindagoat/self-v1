# SelfStorage.help: Website Copy Index

A complete copy deck for selfstorage.help, split into one markdown file per page for easy download, sharing, and editing.

## How to use this folder

Each file is a standalone markdown document. Drop it into your CMS, paste it into Google Docs, or hand it to a developer as-is. File names are numbered so they sort into a sensible reading order in any file browser.

## Voice standard (applies to every file)

- US English
- Direct second person (you, your facility, your units)
- No em dashes
- No rhetorical "not X, Y" contrasts
- No negative references to other agencies or platforms
- Storage-industry vocabulary (move-ins, occupancy, map pack, drive-up, climate-controlled)

See `00-foundation-voice-lexicon.md` for the full voice and lexicon guide before editing any copy.
